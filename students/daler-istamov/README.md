# Birth and Death Rates Analysis Project: Uzbekistan (2010-2024)

This project analyzes population data for Uzbekistan, focusing on births, deaths, and natural growth between 2010 and 2024. It demonstrates a complete data processing workflow, from loading raw datasets to computing derived metrics and visualizing results.

## Features
- Loads data from csv
- Processes loaded data
- Performs certain computations
- Visualizes computed data
